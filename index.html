📖 Tally Import Instructions
For Tally Prime XML Import:
  1. Open Tally Prime → Select your company
  2. Go to Gateway of Tally → Import → Data
  3. Select Vouchers
  4. Browse and select the downloaded XML file
  5. Review and confirm the import
For Tally ERP 9 XML Import:
  1. Open Tally ERP 9 → Select your company
  2. Go to Gateway of Tally → Import of Data → Vouchers
  3. Enter the path of the downloaded XML file
  4. Press Enter to import
Pre-requisites (both versions):
  • Ledger accounts must already exist in Tally (or create them before import)
  • Ensure company name in XML matches your Tally company exactly
  • Suspense Account ledger should be under "Current Liabilities" or "Indirect Expenses"
